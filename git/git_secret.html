配置push免密 1.进入 git 客户端 2.windows上 ssh-keygen.exe 生成公钥 私钥 (第一次git连接github) linux上 ssh-keygen 生成的文件在当前用户.ssh目录 3.将公钥复制到 github ---> setting ---> ssh keys ---> add ssh keys 4.git push origin master (测试是否需要密码) 5.电脑第一次提交时需要config git config --global user.email "caopeng8787@163.com" git config --global user.name "pengfen"
